# IdleActivityBot

## Clone - Build - Install

### 1. Clone repository
```shell
git clone https://github.com/CbICuK/IdleActivityBot.git
```

### 2. Install requirements
```shell
cd ./IdleActivityBot
python -m pip install -r requirements.txt
```

### 3. Build

```shell
python -m PyInstaller .\bot.spec
```

### 4. Run 

```shell
.\dist\work_time_bot\work_time_bot.exe
```
